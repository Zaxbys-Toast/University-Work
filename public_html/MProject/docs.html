Client (package Mproject.client)

Overview

Connects to the server over a socket and manages sending/receiving Message objects (e.g., TextMessage, CommandMessage).

Fields

  • int associateduser

  • Socket socket

  • BoolRef shouldStop

  • static final int defaultport

  • MessageQueue recvmessages

  • HashMap<Integer, ArrayList<Message>> messageLists

Constructor

public Client(String ip, int userId)

  • Connects to the server using IP and port.

  • Sends a CommandMessage to associate the client with a user ID.

  • Starts a ReceiveThread to asynchronously listen for incoming messages.

Methods

Usage Example

xxxxxxxxxx
Client client = new Client("127.0.0.1", 42);
TextMessage msg = new TextMessage("Hello", 42, recipients, groupId);
client.send(msg);

Notes

  • ReceiveThread continuously reads serialized messages from the socket.

  • Messages are stored both in a local queue and grouped by conversation ID.

  • Stopping the client gracefully shuts down the socket.

Server (package Mproject.server)

Overview

Manages client connections, message routing, and server lifecycle.

Fields

  • ServerSocket acceptsocket

  • MessageQueue sendqueue

  • AssociationMap associations

Constructor

xxxxxxxxxx
public Server()

  • Initializes the server with messaging and association maps.

Methods

Usage Example

xxxxxxxxxx
Server server = new Server();
server.start(InetAddress.getByName("0.0.0.0"), 50498);

Notes

  • Uses ConnectionHandler to accept incoming sockets.

  • ReceiveThread reads incoming messages.

  • SendThread delivers outgoing messages to clients.

  • AssociationMap keeps track of connected clients.
